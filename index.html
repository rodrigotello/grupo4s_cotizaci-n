El siguiente proyecto describe el proyecto de capacitación y training sobre Web Development, Web Design y selección de plataformas existentes para desarrollo web para el cliente Grupo 4S, por parte de THINK, la agencia de estrategia y servicios digitales dentro de Manuable SAPI de CV.

Taller 1

Web Development For Managers (& Non Programmers)

Descripción

El Taller "Web Development For Managers (& Non Programmers)" consta de la introducción conceptos y términos sobre cómo está construida la Web y un sitio web.

Temario

  • El Internet
  • Paradigma Cliente - Servidor
  • Protocolos en Internet
  • Dominio vs Hospedaje
  • Página vs Sitio vs App (Web vs Mobile)
  • HTML, CSS, JS: Qué es cada uno y cuál es su rol en el desarrollo web
  • Base de datos: Qué es y por qué es importante
  • CMS

  • Wordpress: Qué es, cómo se usa, arquitectura, cómo se modifica, ventajas y desventajas

  • Tumblr: Qué es, cómo se usa, arquitectura, cómo se modifica, ventajas y desventajas

  • Squarespace: Qué es, cómo se usa, arquitectura, cómo se modifica, ventajas y desventajas

  • Siteleaf: Qué es, cómo se usa, arquitectura, cómo se modifica, ventajas y desventajas

  • Frameworks:
  • HTML / CSS: Qué es, cómo se usa, arqutiectura, ventajas y desventajas
  • Ruby on Rails: Qué es, cómo se usa, arqutiectura, ventajas y desventajas
  • Angular: Qué es, cómo se usa, arqutiectura, ventajas y desventajas

Entregable

Duración:

2 Sesiones por Semana

2 horas por Sesión

8 sesiones en total

Duración: 16 horas (4 semanas)

El taller se llevará a cabo de la siguiente manera:

  • Semana 1

    • Sesión 1:Internet, Paradigma Cliente-Servidor, Protocólos

    Sesión 2:Dominio, Hospedaje, Página, Sitio, App

  • Semana 2

    • Sesión 3:Página, sitio, Web App

    Sesión 4:Dominio, Hospedaje, Página, Sitio, App

  • Semana 3

    • Sesión 5:Wordpress, Tumblr

    Sesión 6:Squarespace, Siteleaf

  • Semana 4

    • Sesión 7:HTML, CSS, JS

    Sesión 8:Frameworks, Ruby on Rails, Angular
